# TravleBuddy

TravleBuddy is an AI-powered travel planner that helps users create personalized itineraries from their preferences, budget, and trip goals. It suggests places, organizes selections into a map-based schedule, detects conflicts, and lets users save or export trips.

## Getting Started

Install dependencies:

```powershell
npm install
```

Run the development server:

```powershell
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
